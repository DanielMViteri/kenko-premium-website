# KENKO Premium

Premium Investments, Powered by Data.

KENKO Premium is a data-driven real estate investment intelligence platform focused on market analysis, climate risk forecasting, and long-term investment strategy.

This repository contains the source code for the official KENKO Premium website.

## Core Areas

- Real Estate Market Intelligence
- Climate Risk Forecasting
- Investment Strategy Analytics
- Data-Driven Research

## Tech Stack

- React
- TypeScript
- TailwindCSS
- Vite
- Lovable.dev
